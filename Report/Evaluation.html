Weekly Review:

Brief:

My project meets all requirements of the brief. It is a stand-alone device and once started it can measure rainfall automatically. An alarm is triggered when the rain sensor records a value less than 1000. The data is sent to Firebase where it is configured in a graph. The rainfall can be monitored from anywhere with an internet connection through the website.

Challenges:

I ran into a few challenges linking my device to Firebase and to the website which I have discussed in my Implementation and Testing Section. I was able to solve any challenges I encountered. I was able to draw on my previous experience of using Firebase and also on online resources and tutorials.

Design Objectives:

I was able to include all elements I had planned for my device in the finished product. I also decided to add an audio alarm I had not previously included as it would make the device more inclusive.
